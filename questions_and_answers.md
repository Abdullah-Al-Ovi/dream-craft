###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: (A) 

<i>As greeting is assigned as an object and there is no property in it.So it will be empty object.</i>

</p>
</details>

###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: (C)

<i>Answer should be "C" as JavaScript performs type coercion and converts the number 1 to a string "1". Then it concatenates the two strings and the result is in string "12".Since there is no console.log.So output will not be printed</i>

</p>
</details>

<details><summary><b>Answer</b></summary>

<i>We are not dealing with food.We are dealing with info.So food remains unchanged</i>

</p>

###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sayHi(name) {
  return `Hi there, ${name}`;
}

console.log(sayHi());
```

- A: `Hi there,`
- B: `Hi there, undefined`
- C: `Hi there, null`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

</p>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: (C)

<i>As first index is 0 which is falsy value so count remains same.Then for index 1,2,3 the condition is considered as true and value of count increases by 1 three time and stands at 3.</i>

</p>
</details>
